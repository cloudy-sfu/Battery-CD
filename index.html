Batteries Charging and Discharging Plan

Batteries group
If maximum input and output power is known:
  • charging duration = energy capacity / maximum input power
  • discharging duration = energy capacity / maximum output power
Supply and demand

Hourly supply and demand electricity load table:

In table "hourly supply and demand electricity load",
  • Format: Accept *.csv file via uploading.
  • Columns: The first column represents the demand load. The second and subsequent columns represent other source of supply load. If there is no other supplies, the table should only have the first column.
  • Column names: Column names won't affect results, but the first row must be column names instead of data.
  • Row names: Row names should not exist.
  • Units: The unit must be corresponding to that of energy capacity of batteries.
  • Order: Each row represents the load of 1 hour, from earliest to latest.
